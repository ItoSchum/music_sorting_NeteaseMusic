# Music Sorting for Netease Music Downloads
## Main Feature
- Sort up **Netease Music**'s downloads by **Artist Names**.

## Usage
1. Change your Teriminal's current directory to `[NeteaseMusic_downloads_dir]`.
2. Run the script.
